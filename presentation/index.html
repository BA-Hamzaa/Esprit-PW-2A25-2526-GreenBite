🏗️ MVC Architecture

📂

Models

Pure PDO — Singleton Database::getConnexion(). No MySQLi. 14 entity classes handling all DB interactions.

⚙️

Controllers

8 controllers routing all business logic: Auth, User, Article, Comment, Nutrition, Marketplace, Recettes, FaceAuth.

🖼️

Views

Separated into Frontoffice (users) and Backoffice (admin). Clean layout system with shared headers/footers.

🌐

Front Controller

index.php is the single entry point — routes all requests via ?page= & ?action= parameters.

🔒 Role-based access: VISITEUR / USER / COACH / ADMIN
🔔 Real-time notification system
✅ Moderation workflow for all content

✨ Key Features

🤖

GreenBot AI

Integrated AI chatbot via OpenRouter API for personalized nutrition advice and meal suggestions.

📊

Admin Dashboard

Chart.js-powered statistics: meal distributions, user activity, order analytics, and content moderation panels.

🌑

Dark Mode

Full dark/light mode toggle with glassmorphism cards, micro-animations, and a premium design system.

💳

Stripe Payments

Server-side PaymentIntent creation. Client-side Stripe.js for secure card collection and order confirmation.

📧

Email System

PHPMailer for transactional emails: password recovery tokens, order confirmations, admin alerts.

🧬

Face ID Login

Biometric authentication using face-api.js — register your face, log in with a look. No password needed.
